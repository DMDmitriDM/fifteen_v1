### Пятнашки (сборка Webpack).
* «Пятнашки» — это популярная головоломка (игра в 15), созданная в 1878 году,
где нужно упорядочить 15 пронумерованных плиток в квадратной коробке 4 X 4,
перемещая их с помощью одной пустой ячейки.
Цель — выстроить числа по возрастанию (1–15)
за минимальное количество ходов, развивая логику и память.

### Необходимые требования
node.js версия не ниже 22.12.0
npm версия не ниже 10.9.0
При необходимости установить serve
npm install --global serve

### Установка и запуск проекта
1. Склонируйте данный репозиторий к себе на диск.
2. Из директории fifteen_v1 выполните команду `npm i` для установки
и npm run dev для запуска проекта в режиме отладки.
Затем откройте браузер по адресу http://localhost:8080/.

3. Для создания продакшен проекта остановите сервер отладки командой CTRL+C
(если он был запущен).
4. Из директории bura выполните команду npm run build.
5. Из директории fifteen_v1 выполните команду npx serve
Затем откройте браузер по адресу http://localhost:3000/.
